# Punish_DMMGamePlayer_for_Umamusume
謎のCPU使用率を誇るDMMGameplayerをウマ娘が起動させたら終了させるだけ

まほうのことば
```cmd.exe
   powershell -NoProfile -ExecutionPolicy RemoteSigned .\openuma.ps1
```
